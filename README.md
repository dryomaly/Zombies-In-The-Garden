# Zombies in the Garden 🌱

Tower Defense в стиле Plants vs. Zombies — учебный проект.

## Описание

Защищайте базу от волн зомби, расставляя растения рядом с дорогой.
Зомби идут по заданному пути (меняется на каждом уровне) и при достижении базы
наносят ей урон. Если "здоровье" базы упадёт до нуля то вы вроиграете.

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

## Управление

| Действие | Клавиша / кнопка |
|---|---|
| Выбрать Стрелка | Q или клик по карточке |
| Выбрать Боксёра | W или клик по карточке |
| Выбрать Пулемёт | E или клик по карточке |
| Поставить растение | ЛКМ по зелёной клетке |
| Снять выбор | ПКМ или ESC |
| Пауза | P |

## Структура проекта (MVC)

```
pvz/
├── main.py                  # точка входа, game loop
├── settings.py              # все константы
│
├── model/                   # игровая логика (Model)
│   ├── game.py              # GameModel — главная модель
│   ├── tile_map.py          # карта тайлов
│   ├── pathfinder.py        # алгоритм A*
│   ├── spatial_grid.py      # пространственное хеширование
│   ├── zombie.py            # зомби + FSM
│   ├── plant.py             # растения + фабрика
│   ├── projectile.py        # снаряды + AABB коллизия
│   ├── wave_manager.py      # волны + weighted random
│   └── base.py              # база игрока
│
├── view/                    # отрисовка (View)
│   ├── renderer.py          # основной рендер
│   ├── draw_entities.py     # рисуем зомби и растения
│   └── ui.py                # экраны меню
│
├── controller/              # ввод (Controller)
│   └── input_handler.py     # обработка событий pygame
│
├── data/
│   ├── levels/              # JSON-файлы уровней
│   │   ├── level1.json
│   │   ├── level2.json
│   │   └── level3.json
│   └── scores.json          # сохранённые рекорды (создаётся автоматически)
│
└── tests/
    └── test_model.py        # юнит-тесты (pytest)
```

## Алгоритмы

| Алгоритм | Файл |
|---|---|
| Game Loop + deltaTime | main.py |
| Обход пути зомби по waypoints | model/tile_map.py |
| Взвешенный случайный выбор | model/wave_manager.py |

## Типы растений

| Растение | Стоимость | Урон | Радиус | Атак/сек | Снаряд |
|---|---|---|---|---|---|
| Стрелок | 100 | 10 | 3.5 | 1.0 | Да |
| Боксёр | 150 | 25 | 1.2 | 1.5 | Нет (мгновенно) |
| Пулемёт | 200 | 12 | 5.5 | 2.5 | Да |

## Типы зомби

| Зомби | Скорость | HP | Урон по базе |
|---|---|---|---|
| Обычный | 0.55 тайл/с | 100 | 25 |
| Быстрый | 1.6 тайл/с | 40 | 10 |
